Experiencia Profesional

Encriptador de Texto

Desarrollé una página de prueba que permite encriptar y desencriptar texto utilizando un algoritmo básico de cifrado. La interfaz es sencilla e intuitiva, permitiendo a los usuarios ingresar texto, ver el resultado encriptado y copiarlo con un solo clic. Este proyecto fue creado con JavaScript, HTML y CSS como parte de mi práctica en desarrollo web.

Juego Secreto

Creé un juego interactivo en el que los usuarios deben adivinar un número secreto generado aleatoriamente dentro de un rango específico. Los jugadores reciben pistas si el número es mayor o menor, hasta acertar. El proyecto fue desarrollado con JavaScript, HTML y CSS, y está diseñado para mejorar la lógica y habilidades de deducción de los jugadores.
